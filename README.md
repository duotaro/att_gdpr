## Cocoapods
add IronSourceSDK
```ruby:Podfile
    pod 'IronSourceSDK','8.4.0.0'
```

When using Google User Messaging Platform to comply with the General Data Protection Regulation (GDPR), please also add the following.
```ruby:Podfile
    pod 'GoogleUserMessagingPlatform'
```
